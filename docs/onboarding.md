# Onboarding

1. Read `CONTEXT.md` for business requirements.
2. Copy `.env.example` to `.env` and fill local values.
3. Install the package with `pip install -e .`.
4. Run `customer360 healthcheck`.
5. Run `pytest` before opening a pull request.
6. For containerized development, run `docker compose build` and then follow
   `docs/containerization.md`.
